# Simscape Model of 10-DOF VTOL Drone 🛠️

This module contains the physical simulation of the VTOL drone using MATLAB Simscape Multibody.

## Objective:

To build a realistic 3D physical model of the drone and simulate its motion under different inputs.

## Model Description:

* Built using Simscape Multibody
* Includes drone frame, rotors, and joints
* Actuation through thrust inputs
* Realistic constraints and physical interactions

## Features:

* 3D visualization of motion
* Multi-body dynamics
* Real-time simulation
* Accurate representation of mechanical structure

## Components Used:

* Rigid bodies
* Revolute joints
* Actuators (rotors)
* Sensors (for state measurement)

## Files:

* `simscape_model.slx`

## Simulation Output:

* Position and orientation tracking
* Motion visualization
* Dynamic response of the system
